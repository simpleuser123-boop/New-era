## 1. 设计系统：「暖纸手帐」

### 创意北极星："一页翻开的手帐"

- 有机圆角体系（8px → 24px，随层级上升变圆）
- 温暖的琥珀色承担界面 30-40% 的视觉重量
- 拒绝一切"招聘 App"刻板印象——不用表格堆砌、不用企业蓝

## 3. API 路由


## 5. 禁止事项

- 禁止纯白 `#ffffff` 作为页面背景
- 禁止纯黑 `#000000` 作为文字颜色
- 禁止高饱和度色彩（荧光蓝、荧光绿）
- 禁止深色阴影
- 禁止直角 — 所有元素至少 8px 圆角
- 禁止表格堆砌信息
- 禁止信息密度过高
- 禁止 emoji 作为主要视觉元素（用 Lucide 图标）
- 禁止渐变背景
- 禁止粗分割线
