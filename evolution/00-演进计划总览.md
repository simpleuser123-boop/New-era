# New Era — 演进计划总览

## 产品愿景

> 从"海投焦虑"到"精准掌控" — 为 AI 从业者打造的专属求职助手，让每个 AI 岗位都值得认真对待。

## 项目定位

New Era 是一个面向中国市场的 **AI 方向专属求职助手**。不做通用求职工具，聚焦 AI 从业者的真实痛点：
- 这个 AI 岗是真的吗？（假 AI 岗检测）
- 这个岗在 AI 生态的什么位置？（LLM 应用 / AI Infra / 算法研究 / 具身智能）
- 我的技能栈匹配这个方向吗？（AI 技术栈映射）
- 这个方向的前景如何？（AI 行业趋势洞察）

## 项目现状（2026-06-11）

New Era 已从早期 **竞赛 Demo 前端原型** 演进为 **local-first 个人 AI 求职工作台**。当前代码已接入 SQLite 本地持久化、owner 单用户鉴权、Tracker、Career DNA、Analytics、Interview Coach、Resume Judge、Agent 助手，以及围绕“真 / 假 AI 岗识别”的风险 benchmark、真假对决和 HR 核实问题。

仍需保持的产品边界：
- 不接招聘平台授权，不代替用户投递，不抓取登录态招聘网站。
- 不做多租户 SaaS、公开分享、支付订阅或云端同步。
- Agent 的 Query 工具只读；Action 工具只能创建待确认动作，不直接写业务表。

```
当前主线（local-first）
──────────────────────────────────────────────────────────────
首页仪表盘           JD 评估 + 报告          AI 岗位评估 6 维模型
风险扫描             真假 AI 岗对决          风险 benchmark / 反馈
简历管理             Resume Judge / ATS      简历解析 / 匹配 / 优化
投递管理             Tracker 时间线          本地投递漏斗
Career DNA           求职画像建议            本地画像设置
求职分析             Analytics 聚合          漏斗 / 方向 / 风险 / 技能缺口
AI 助手              多 Agent / SSE          会话 / 可控记忆 / 待确认动作
面试教练             问题 / 回答 / STAR      面试会话持久化
行业趋势             趋势 / 周报             预置数据 + DeepSeek 降级
登录注册             owner 鉴权              密码 / session / 受保护页面
设置                 导出 / 清除 / 密码       隐私与数据边界说明
```

### 规模概览

| 维度 | 当前实现 | 后续规划 |
|------|:---:|:---:|
| 前端页面 | 15 个 `page.tsx`（13 个主产品页 + 2 个辅助页） | 继续按单机产品补体验闭环 |
| API 路由 | 54 个 route 文件 | 按功能补 smoke 和错误契约 |
| Agent 工具 | Query Tools + Action Proposals | 扩展更多只读分析和待确认动作 |
| 核心引擎 | 评估 / 风险 / 匹配 / 画像 / 面试 / Agent | 强化验真质量与演示稳定性 |
| 数据库表 | 17 张主表 / 支撑表 | 继续 additive migration |
| 数据来源 | 本地 SQLite + demo-data + DeepSeek | 可选导入，不做平台授权 |
