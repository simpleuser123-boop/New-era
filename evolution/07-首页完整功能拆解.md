## 功能拆解

### 1. 每日问候语 `getGreeting()`

```typescript
function getGreeting(): string {
  const hour = new Date().getHours();
  const day = new Date().getDay();
  const dayNames = ["周日", "周一", "周二", "周三", "周四", "周五", "周六"];
  if (hour < 10) return `早安，${dayNames[day]}愉快`;
  if (hour < 14) return `午安，${dayNames[day]}顺利`;
  if (hour < 19) return `下午好，${dayNames[day]}加油`;
  return `晚安，${dayNames[day]}辛苦了`;
}
```

副标题每日轮换 7 条求职格言。

### 2. Hero 指标卡

| 指标 | 计算方式 |
|------|---------|
| 已评估岗位 | `reports.length` |
| 高匹配岗位 | `reports.filter(r => r.score >= 4.0).length` |
| 风险拦截 | `reports.filter(r => r.risk_level === 'high' || r.risk_level === 'critical').length` |
| 市场活跃度 | 预置数据（"活跃"/"稳定"/"冷淡"） |

每个指标显示周环比趋势。

## 数据加载流程

```
用户访问 /
  ├─ reports 数据: GET /api/reports → 前端渲染
  ├─ applications 数据: GET /api/applications/stats → 投递漏斗
  ├─ AI 快讯: 预置数据或 DeepSeek 生成，必须标记来源
  └─ 热门技术栈: 预置数据
```
